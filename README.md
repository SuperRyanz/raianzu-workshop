
Arman Jaya - Frontend v1.0

Template frontend statis untuk Arman Jaya (Website Bengkel & Penjualan Onderdil).
Dibuat dengan Bootstrap 5 + custom CSS. Siap dikembangkan ke backend (MySQL / Node.js / Flask).

Cara pakai:
1. Download dan ekstrak project.
2. Buka index.html di browser atau pakai local server.
3. Ganti gambar di assets/img sesuai kebutuhan.
